Editkaro Portfolio

A modern landing page for Editkaro.in, a premium video editing agency. This project showcases responsive navigation, service sections, portfolio cards, animated counters, reveal-on-scroll effects, and a YouTube video modal.

 Features

- Responsive hero section with call-to-action buttons
- Animated scroll reveal for page sections
- Animated stats counters
- Client testimonials section
- Portfolio filter buttons for content categories
- Mobile-friendly hamburger menu and mobile navigation
- Video modal popup for portfolio cards
- Footer with branding and company copyright

 Built With

- HTML
- CSS
- JavaScript
- Font Awesome icons
- Google Fonts (`Space Grotesk`, `Inter`)

 Project Structure

- `index.html` — main landing page markup
- `style.css` — site styling and responsive layout
- `script.js` — interactive behaviors for navigation, animations, filters, and modal

 Usage

1. Open `index.html` directly in a web browser.
2. Alternatively, serve the folder with a local development server if you want clean URLs and live reload support.

 How to Run Locally

 Option 1: Open file directly

- Double-click `index.html` or open it in your browser.

 Option 2: Use a local server

If you have Node.js installed, run a simple static server from the project folder:

```bash
npx http-server .
```

Then open the provided local URL in your browser.

 Customization

- Update the hero text, portfolio cards, and service copy in `index.html`.
- Change colors and typography in `style.css`.
- Adjust animation timing and modal behavior in `script.js`.

 Notes

- The project is designed as a static portfolio landing page.
- Make sure `style.css` and `script.js` remain linked correctly in `index.html`.
